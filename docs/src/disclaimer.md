# Disclaimer

Tutela is an **unofficial** client built from publicly published OpenAPI
documentation. It is not affiliated with, endorsed by, or supported by Guardian.

Two things the published specs don't include and that you must supply yourself:

- the real OAuth **token URL** (the specs ship a placeholder), and
- your partner-issued **client credentials and API key**.
